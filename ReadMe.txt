Welcome

For anyone who wants to manage his library you have chosen the right place.

Here you can act as a library administrator to manage books in your library 
and to add, edit, search for books.
 
How it works

First the app uses excel file to read library books,
then show choices for librarian to choose the action he needs

1)View All books
2)Add a book
3)edit a book
4)search for a book
5)exit app

1)View all books
show all the books with the ability to show specific book data

2)Add book
by adding book title and book author as required fields
and book description

3)edit book
edit book title, author and description

4)search book
you can search book by part of it's title
and then choose the book you need to know it's data

5)exit app
